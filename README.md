# Savory Skies - Modern Restaurant MERN Application

Savory Skies is a premium, fully functional, and interactive restaurant website built with the MERN stack. Designed for a seamless user experience, it features a dynamic menu, real-time reservations, a stunning gallery, and a comprehensive admin dashboard.

## 🚀 Objective
Create a professional, mobile-friendly, and visually appealing online presence for a local restaurant business to attract customers and streamline operations.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite), Tailwind CSS, Framer Motion, Lucide React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT (JSON Web Tokens)
- **State Management:** React Context API / Redux Toolkit
- **Notifications:** React Hot Toast / Sonner

## ✨ Core Features
- 🏠 **Home Page:** Hero section, featured dishes, about preview, and testimonials.
- 📜 **Menu Page:** Categorized items with search, filters (veg/non-veg, price), and real-time updates.
- 📖 **About Page:** Storytelling section with chef introductions and mission.
- 🖼️ **Gallery Page:** Masonry layout with lightboxes and lazy loading.
- 📞 **Contact Page:** Integrated Google Maps, contact form, and business hours.
- 📅 **Reservations:** Interactive booking system with real-time availability and email confirmation.
- 🔐 **Admin Dashboard:** Manage menu items, view reservations, and handle contact queries.

## 🎨 Design Principles
- **Aesthetics:** Warm, premium tones with modern typography (Outfit/Inter).
- **Interactivity:** Smooth scrolling, hover animations, scroll-based fade-ins.
- **Responsiveness:** Mobile-first approach for all devices.
- **Performance:** Optimized assets and efficient API handling.
